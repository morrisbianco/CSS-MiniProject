Andrew's CSS Snippet Cheatsheet

My Personal Cheat Sheet of my six favorite pieces of CSS Code! With this I can never fail to write out these codes again!

N-th-Child

The N-th-Child ( ) psuedo code selects every child tag that you have specified within the parentheses, such as even or odd children. 


                :n-th-child(even) {
                    border-color: green;
                }

:root

The :root code is placed at the top of the CSS sheet and is used to create variables that will be used throughout the entire style sheet. 


                :root {
                    --dark: black;
                    --light: white;
                    --text-color: green;
                }

Data-Descr

The Data Description attribute is used to show more information about a section on the page through the use of span tag and data-descr=" ". 

                
                span [data-descr] {
                    content: attr(data-descr);
                    position: absolute;
                    cursor: pointer;
                }

Transform: Rotate

The Transform: Rotate ( ) code allows you to rotate an image any set of degrees that you place in the parentheses. /p> 


                img {
                    transform: rotate(90deg);
                }

Flex

The expanded flex tag allows the section to grow or shrink as the screen size is adjusted. In the code example, the section can grow as the screen changes but not shrink. The third number marks the minimum size for the flex-wrap. 

                
                section {
                    display: flex;
                    flex-wrap: wrap;
                    flex: 1 0 200px;
                }

Linear-Gradient

The Linear-Gradient code allows you to merge to colors into one color through a gradient shift between the colors. By using background, the colors will target the background of the class.


                .box1 {
                background: linear-gradient(limegreen,lime);
                }
